Question:

Create a hierarchy of classes for a set of vehicles using the following class names: Vehicle, Bicycle, Car, Motorbike, airplane.

The common methods for these vehicles are move(), printName(), printType() and printSeat(). The move() method prints out the distinct way on how a specific vehicle moves. For instance, the bicycle will be ‘paddling…’, the airplane will be ‘flying…’. The printName() method prints out registration number of vehicle (e.g. MH371). The printType() method prints the class name. The printSeat() method prints out how many seats available for passengers and drivers for a specific vehicle. The seatSize is a common attribute for all vehicles.

Create a test class to apply the Polymorphism concept using the following scenario.

In the test2 method, use a For loop to loop through the array to toggle the autopilot mode for airplanes only.

Memory partition in Java

2020-04-01 by Domuki

Java memory is mainly divided into five parts:

  1. Stack: the stack is used to store the local variable in methods({}).
  2. Heap: the heap is used to store the thing which created by “new”, and everything has their own reference and default value.
type default value
int 0
double 0.0
char ‘\u0000’
Object null
boolean false
  1. Method area: the information related to .class (include method information)
  2. Native Method stack: Operation System related
  3. PC Register: in CPU, very effective
