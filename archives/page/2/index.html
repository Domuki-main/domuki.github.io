User Interface and User Experience Week 5

2021-05-06 by Domuki

Presentation

Group information

Project title: Color it

Group name: WeDesign

Group member:

  • Aous
  • Ma
  • Wang

Project/Research information

Introduction

Designer struggling to find a color that fits their needs.so designers can select their own appropriate color groups or individual colors for use in the designing their software, or generate random colors automatically.

Objective

To help designers find the color that is inside their mind. Facilitate designers to design a more beautiful UI interface.

Background

For the designers who are not familiar with the colours code, it is hard to design and develop the website or application interface without a sample of colour code. Many people who do not know much about design and have not studied systematically find it difficult to use the right color combinations to make good-looking UI designs.

In order to help designers and developers to find the colours they want, this website will provide both code and the sample of the colours.

Methods

The website will have an option where it displays different types of colors and the user can come and choose which color fits him best.

Expected Result

The designer finds the colors for their needs

Implementation

  • Figma
  • HTML and CSS with Bootstrap framework

Reference

https://htmlcolorcodes.com/

04/05/2021

Empathize, the first stages of Design thinking process.

The first stage of the design process is to understand the perspective of the target audience/customer/consumer to identify and address the problem at hand.

Asking yourself these three questions is a good way to start practicing empathy in your everyday interactions:

  • What is this person feeling?
  • Can you identify this feeling?
  • How would you describe about felt this way?

Design thinking is a term used to represent a set of cognitive, strategic and practical processes by which design concepts are developed.

Design thinking is a good way for students to do their design project more efficient.

Empathy map can be used in many situations in order to inspire designer to solve problems. You can use empathy maps to summarize what you’ve learned from your interactions with people in the field of design studies.

06/05/2021

A journey map is a synthetic representation that describes a step-by-step description of how a user uses a service.

journey map is the process of providing a complete picture of the customer experience by revealing moments of both frustration and joy through a series of interactions. When executed successfully, it opens up opportunities to solve customer problems, reduce fragmentation and ultimately improve the user experience.
