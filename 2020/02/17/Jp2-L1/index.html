​ • A method named getPerimeter() that returns the perimeter.

​ • A no-arg constructor Rectangle().

​ • A constructor with parameters that accepts width and height values: Rectangle(width: double, height: double).

Write a test program that creates three Rectangle objects:

​ i. using no-arg constructor with default values of 1 (for both)

​ ii. using no-arg constructor with assigned values: width = 4 and height = 40

​ iii. using argument constructor that accepts width and height values from the user.

Display the width, height, area and perimeter of each rectangle in this order.
