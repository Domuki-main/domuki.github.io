Question 1

Write a program that prompt the user to enter the length in feet and inches and outputs the equivalent length in inches and in centimetres. If the user enters a negative or a non-digit number, throw and handle an appropriate exception and prompt the user to enter another set of numbers.
Note: 1 foot is equal to 12 inches; 1 inch is equal to 2.54 centimetre. You may use InputMismatchException and NumberFormatException classes.
