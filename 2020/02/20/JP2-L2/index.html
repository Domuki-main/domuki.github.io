Question 2

  1. Write a class named RetailItem that holds data about an item in a retail store. The class should have the following properties:

    • The description field that references a String object that holds a brief description of the item

    • The unitsOnHand field is an int variable that holds the number of units currently in inventory

    • The price field is a double that holds the item’s retail price

    • A constructor that accepts arguments for each field, appropriate mutator methods that store values in these fields, and accessor methods that return the values in these fields.

  2. Write a CashRegister class that can be used with the RetailItem class above. The CashRegister should simulate the sale of a retail item. The class should have the following properties:

    • A constructor that accepts a RetailItem object as an argument and an integer that represents the quantity of items being purchased

    • The getSubTotal method that returns the subtotal of the sale, which is the quantity multiplied by the price. This method must get the price from the RetailItem object that was passed as an argument to the constructor

    • The getTax method that should return the amount of sales tax on the purchase. The sales tax rate is 6 percent of a retail sale

    • The getTotal method that returns the total of the sale, which is the subtotal plus the sales tax.
