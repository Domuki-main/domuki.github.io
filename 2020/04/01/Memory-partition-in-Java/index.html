Memory partition in Java
Domuki

  • | Tips 
 

Java memory is mainly divided into five parts:

  1. Stack: the stack is used to store the local variable in methods({}).
  2. Heap: the heap is used to store the thing which created by “new”, and everything has their own reference and default value.
type default value
int 0
double 0.0
char ‘\u0000’
Object null
boolean false
  1. Method area: the information related to .class (include method information)
  2. Native Method stack: Operation System related
  3. PC Register: in CPU, very effective
