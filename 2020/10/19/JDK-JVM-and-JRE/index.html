JDK, JVM and JRE

Overview

We must understand the differences between JDK, JRE, and JVM before proceeding further to Java.

Brief Introduction

JVM

JVM (Java Virtual Machine) is an abstract machine.

It doesn’t physically exist.

It provides a runtime environment in which Java bytecode can be executed.

It can also run those programs which are written in other languages and compiled to Java bytecode.

JRE

JRE is Runtime Environment.

A set of software tools which are used for developing Java applications.

It is used to provide the runtime environment.

It is the implementation of JVM.

It includes a set of libraries and other files that JVM uses at runtime.

JDK

JDK(Java Development Kit)

is a software development environment which is used to develop Java applications and applets.

It physically exists.

It contains JRE and development tools.
