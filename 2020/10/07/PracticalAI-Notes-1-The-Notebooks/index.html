PracticalAI Notes 1: The Notebooks

Author: Domuki Date: October 7, 2020  18:40:29 Category: Artificial Intelligence

Introduction of PracticalAI

PracticalAI is a totally free project or tutorial based on Google Cloud of deep learning on GitHub platform.

Preparation

1. Download

Click the LINK, then download the .zip on GitHub.

2. Run the Files

  1. Get the notebooks from the notebooks folder in this repositories.

  2. For running the files, we need run these notebook on Google Colab (recommend) or your location machine.

  3. Before following operations, sign into your Google account.

  4. Click on a notebook and replace https://github.com/ with https://colab.research.google.com/github/ in the notebook URL or use this Chrome extension to do it with one click.

    Or, you can just download all the files to your local machine and upload to your Google Colab.

  5. Run the files. and all changes will be automatically saved to you personal Google Cloud.

How to work with Notebook

If you already know how to use notebooks, just skip this section.

Cell Types

Text cells: used for text

Code cells: used for code

(literal meaning)

Operations

Creating

Just click on +TEXT or +CODE button below the View button up top.

Running

Press SHIFT + ENTER to run the cell.

Editing

Double click

Moving

Put your spot on the pane of the cell and you can see that there is a toolbar on the right-top corner. Press ⬆️ or ⬇️ buttons to change your cell’s position.

Deleting

Press the button with a garbage can icon on the toolbar.

Saving

Press Ctrl + S
