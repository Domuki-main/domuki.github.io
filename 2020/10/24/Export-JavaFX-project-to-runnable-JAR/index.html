Exporting

  1. Open Eclipse.
  2. Click “File“ on the top left selection bar.
  3. Click “Export“
  4. Select Runnable JAR file, and next
  5. Choose Launch configuration and Export destination
  6. Then Finish.

Problems

You may get the problem like this:

When you double click the JAR, it doesn’t response you.

Then you may use CMD to run this stupid file using java -jar <filename>
