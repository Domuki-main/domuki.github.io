Author: Domuki Date: October 6, 2020  18:18:47 Category: Data Structure and Algorithm

Understanding Linked List

A linked list is a linear data structure where each element is a separate object. Each element (we will call it a node) of a list is comprising of two items - the data and a reference to the next node.

Simply,

a linked list consists of nodes where each node contains a data field and a reference(link) to the next node in the list

Working with Linked Lists

Lookup:

  • by value O(n)
  • by index O(n)

insert:

  • at the end O(1)
  • At beginning O(1)
  • in the middle O(n)

Delete:

  • From beginning O(1)
  • from end O(n)
  • from middle O(n)
