Scheduling Algorithms 2 – Round-Robin (RR)

Implement a Java program to calculate the waiting time and turnaround time involving N processes using Round-Robin scheduling algorithm. The user should be allowed to enter the following information:

· N, or the total of processes involved (N should be between 4 to 10 inclusive)

· Burst time for each process

· The quantum time to be used by the RR algorithm.

Arrival time is not needed as it is assumed all processes arrived at the same time (t=0). Using the information provided, calculate

· Waiting times for all the processes

· Turnaround times for all the processes

· The average waiting time produced by the RR algorithm for the given situation/problem.

User input:

No. of Processes = 4
