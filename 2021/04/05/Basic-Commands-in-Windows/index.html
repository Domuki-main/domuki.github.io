Basic Commands in Windows

Author: Domuki Date: April 5, 2021  8:07:39 Category: Operating System

dir - Displaying Directory Content

*cd* - change to the root directory

cd [dir-name] - change to the [dir-name] directory

dir . - Displaying current directory content

dir .. - Displaying parent directory content

dir [file-name] - Displaying current directory content

*dir c:\mydir | find “file1.txt”* - determine the existence of a file named ‘file1.txt’ in a directory named ‘mydir’ in the upper level

md [dir-name] - Create an [dir-name] directory

copy con [file-name] - create an file

copy [file1] [file2] - Make a backup of the file2.doc file with other name file1.doc

ren [file1] [file2] - rename file1 to file2

del [file] - delete a file

rd [directory_name] - Delete a void directory

rmdir [directory_name] - Delete a directory
