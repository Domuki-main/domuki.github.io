User Interface and User Experience Week 6

Author: Domuki Date: May 12, 2021  17:46:31 Category: UIUX

User-centric design care about more details of users and their habits. Human-centric design emphasizes the importance of the human experience in solving problems.

UX and UI are closely related.

UX design is always conducted in first step in producing process, before the step of UI design. UX designers draw user journey maps and basic framework of the users which will be improved with visual and interactive elements by UI designers.

Since UX performed before UI, we need to solve the problem by using people-oriented design. Designers must firmly grasp the requirements of users. They have to ask for user experience questions.

Designers need to build the best user experience for getting a good user interface.

Design thinking is an iterative process in the creative problem-solving process.

Design thinking is important because it helps to create the right environment for a practical and broad understanding of the customer’s voice.
